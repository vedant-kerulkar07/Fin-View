<h1>💰 Fin-View – Personal Finance Management Dashboard</h1>

A modern and responsive personal finance management web application built using the MERN Stack.
Fin-View helps users track income & expenses, visualize spending habits, and gain better control over their finances through interactive dashboards and charts.

<h2>🚀 Features</h2>

👤 Secure User Authentication

Sign up & login with encrypted passwords

User-specific data protection

📊 Interactive Dashboard

💵 Total Income

💸 Total Expenses

🧮 Remaining Balance

📆 Monthly financial summary

➕ Income Management

Add income with amount, source & date

Categorize income (Salary, Freelance, Business, etc.)

View complete income history

<h4>➖ Expense Tracking</h4>

Add expenses with title, category & date

🛒 Food | 🚗 Travel | 🏠 Rent | 🛍️ Shopping | 💡 Bills

Edit & delete expense records

📈 Visual Analytics & Charts

📊 Bar charts for monthly income vs expenses

<h4>🥧 Pie charts for category-wise expenses</h4>

Easy-to-understand financial insights

<h4>📜 Transaction History</h4>

Chronological list of all transactions

Filter by type, date & category

<h4>📱 Responsive Design</h4>

Works smoothly on desktop, tablet & mobile

Clean, modern, and user-friendly UI

<h4>🔐 Data Security</h4>

Password hashing

Protected APIs

User-isolated financial records
